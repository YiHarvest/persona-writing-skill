# 画皮：按抽象风格规律重写

读取立骨结果和选定人物卡，生成完整初稿。

## 1. 选择开头

从人物卡的开头模式中选择与原文内容相符的一种：

- 有可靠数据：数据钩子。
- 有明确事件：事件切入。
- 需要共鸣：场景或痛点切入。
- 需要思考：问题或反常识切入。

没有对应素材时，不为套模式虚构数据、热点或个人经历。

## 2. 选择结构

优先选择与原文逻辑最接近的卡片结构；卡片没有完全匹配的结构时，保留原文逻辑，只迁移节奏和表达方式。不得为了“像”而打乱因果或删除必要限定。

## 3. 重建节奏

参考人物卡的段落比例、信息密度、情感曲线和转折习惯分配内容。比例是弹性参考，不机械凑字数。

## 4. 迁移语言风格

- 迁移长短句、设问、排比、比喻、口语度、专业度和情感温度。
- 把普通表述改造成符合人物卡规律的新表达。
- 口头禅只在自然且不造成冒充时少量使用。
- 不复制人物卡经典金句，不拼接其原句；学习公式并原创。
- 不使用人物卡人物的第一人称履历、家庭、职业或见闻，除非这些也真实属于用户且由用户提供。
- 不把卡片人物的署名、关注口播或身份标识带入成稿，除非用户明确要求并有权使用。

## 5. 事实与边界打磨

- 保留原文事实、数据、时间、因果和限定语。
- 不新增未经证实的事实、来源、引语、案例或统计。
- 不将观点写成事实，不把相关性写成因果。
- 对明显性别化、群体化或攻击性表达，按用户意图做必要中立化，同时保留观点力度。
- 涉及高风险或时效性事实时，以核验结果为准；未核验则不扩写。

## 初稿完成标记

```markdown
✅ 画皮完成：采用「{结构}」与「{开头类型}」，迁移了{关键风格特征}
```
