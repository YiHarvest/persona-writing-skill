# 动嘴：最终质量优化

## 一、先判断内容质量

在调整字数前评估观点完整性、论据充分性、逻辑严密性和节奏紧凑性：

- **完美紧凑型**：只做必要的风格和口播优化，保持长度。
- **内容缺失型**：只补充可由用户材料支持或已经核验的信息；参考卡片长度但不注水。
- **冗长重复型**：删除重复、压缩过渡，保留全部核心观点和有效证据。

## 二、优化检查

1. **开头**：前 50 字是否准确且能抓住注意力，是否存在无依据的夸张。
2. **节奏**：信息密度、情感起伏、转折和段落长度是否符合卡片且适合载体。
3. **表达**：关键观点是否有记忆点，原创表达是否自然，是否误用了卡片原句。
4. **结尾**：是否完成观点闭环；行动号召是否符合用户身份，是否误带卡片人物署名。
5. **事实**：数据、案例、引语、时间、人物和因果是否与原文或核验结果一致。
6. **身份**：是否虚构用户经历或冒充卡片人物；如有，立即删除或改为非第一人称。

## 三、字数策略

`内容质量 > 字数匹配`。人物卡字数只作参考：

- 宁精勿水，不为达到中位数补废话。
- 宁短勿长，不因模仿而重复观点。
- 原文确需补充但无可靠材料时，保留缺口或请求材料，不编造。

## 四、可选 A/B 版本

仅当用户要求、受众明显分化或两种方向都合理时生成：

- A：更重情感和故事。
- B：更重数据和逻辑。

否则只给一个经过判断的最佳版本。

## 五、终检清单

- [ ] 原文核心观点和必要限定均保留。
- [ ] 选定卡片的结构、语气和节奏可辨认，但没有复制标志性原句。
- [ ] 没有新增无依据事实或第一人称经历。
- [ ] 字数服务内容，适合指定载体。
- [ ] 标题、标签和正文与用户目标一致。
